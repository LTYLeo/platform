Getting Started with TAI API

Welcome to the TAI Developer Platform! This guide will help you get started with our API and integrate TAI models into your applications.

Authentication

All API requests require authentication using your API key. You can find your API keys in the Dashboard. 

import tai

# Initialize the client with your API key
client = tai.Client(api_key="YOUR_API_KEY")

Making Your First Request

Here's a simple example of how to use the chat completions endpoint: 

response = client.chat.completions.create(
  model="gtc-2-turbo",
  messages=[
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "Hello, how are you?"}
  ]
)

print(response.choices[0].message.content)

Available Models

TAI offers several models optimized for different use cases:

Model Description Best For
GTC-2 Turbo Our most powerful general-purpose model Complex reasoning, code generation
Giggle (Teens) Fun, engaging personality Entertainment, casual conversation
Bro (Teens) Friendly, supportive companion Mental health, emotional support
Sweetie (Teens) Warm, caring personality Companionship, empathetic responses
